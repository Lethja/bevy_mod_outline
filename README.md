# bevy_mod_outline

This crate provides a plugin for drawing outlines around meshes.

## Dependency

```toml
[dependencies]
bevy_mod_outline = "0.1"
```

## Example

A rotating rounded cube with an outline.

```shell
cargo run --example cube
```
